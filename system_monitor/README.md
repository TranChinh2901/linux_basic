# 📊 Giám sát Hệ thống

## Kiểm tra dung lượng

| Lệnh | Mô tả |
|------|-------|
| `df -h` | Xem dung lượng ổ đĩa |
| `du -sh folder` | Xem dung lượng thư mục |

## Thông tin hệ thống

| Lệnh | Mô tả |
|------|-------|
| `uptime` | Thời gian hoạt động hệ thống |
| `free -h` | Kiểm tra RAM |
| `uname -a` | Thông tin hệ điều hành |

---
> 💡 **Tip:** Dùng `watch -n 1 free -h` để theo dõi RAM real-time!
