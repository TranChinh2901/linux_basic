# Linux Command Guide

Chào mừng bạn đến với hướng dẫn sử dụng các lệnh Linux cơ bản!

## Cách sử dụng
Mở từng folder và đọc nội dung để học các lệnh Linux theo từng chủ đề:

- 📁 **manage_folder_navigation** - Quản lý thư mục và điều hướng
- 📄 **manage_file** - Quản lý file
- 🗜️ **compression_and_extraction** - Nén và giải nén
- ⚙️ **manage_process_service** - Quản lý tiến trình và dịch vụ
- 🌐 **network_and_connect** - Mạng và kết nối
- 🔐 **permission_and_ownership** - Quyền và sở hữu
- 📊 **system_monitor** - Giám sát hệ thống

Chúc bạn học tập hiệu quả! 🚀
